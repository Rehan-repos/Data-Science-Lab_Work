# Control Statements & Data Analysis (Facebook Dataset)

## Description

This lab focuses on analyzing a Facebook dataset using Python. It compares different file formats (CSV and Excel), performs data examination, descriptive statistics, and visualization.

## Dataset

* **Source:** Facebook Friends Dataset (CSV and Excel files)
* **Description:** Contains user-related features such as age, number of photos, tags, posts, likes, and friends count.

## Steps Performed

1. Data Access

   * Loaded both CSV and Excel datasets
   * Compared structure and columns

2. Data Examination

   * Displayed head and tail of dataset
   * Checked data types
   * Verified missing values

3. Descriptive Statistics

   * Used describe() to analyze mean, standard deviation, and distribution

4. Data Visualization

   * Generated histograms for selected numeric features
   * Analyzed data distribution

## Results

* Both CSV and Excel files contain similar data, but Excel preserves structure better
* Dataset has no missing values
* Data shows variation in features like photos, tags, and friends
* Visualization helped understand distribution patterns

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

## Conclusion

The lab demonstrates how different data formats can be handled in Python and how exploratory data analysis helps in understanding dataset structure and patterns.
