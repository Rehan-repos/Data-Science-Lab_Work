# Data Science Lab Work

## Description

This repository contains all lab work and projects completed as part of the course **DATA.110.HY1: Introduction to Python Programming for Machine Learning (Spring A 2026)**. It includes fundamental Python labs, data analysis tasks, and machine learning projects.

---

## Repository Structure

### Labs

* **Lab1_Introduction_to_Python** – Basics of Python programming
* **Lab2_Control_Statements** – Data analysis using CSV and Excel datasets
* **Lab3_NumPy** – Numerical computing using NumPy
* **Lab4_Pandas** – Data manipulation and analysis using Pandas
* **Lab5_Regression_Analysis** – House price prediction using regression models
* **Lab6_KNN_Classification** – Classification using K-Nearest Neighbors
