# Rock Paper Scissors Game (GUI)

## Description

This project is a graphical implementation of the classic Rock-Paper-Scissors game using Python. It allows the user to play against the computer through an interactive GUI built with Tkinter.

## Dataset

* No dataset required
* The game uses predefined choices: Rock, Paper, Scissors

## Steps Performed

1. Created GUI using Tkinter
2. Designed interface with buttons and labels
3. Implemented game logic for Rock-Paper-Scissors
4. Generated random computer choices using Python
5. Displayed results dynamically on screen

## Results

* User can play the game interactively
* Computer generates random moves
* Game displays winner (User / Computer / Tie)
* GUI enhances user experience

## Tools Used

* Python
* Tkinter
* Random module

## Conclusion

This project demonstrates how Python can be used to build interactive GUI applications. It combines logic building with user interface design in a simple and engaging way.
