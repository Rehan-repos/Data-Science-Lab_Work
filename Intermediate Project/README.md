# House Price Prediction using Machine Learning

## Description

This project focuses on predicting house prices using machine learning techniques. It analyzes various housing features such as number of rooms, crime rate, and other factors to estimate property prices.

## Dataset

* **Source:** Boston Housing Dataset
* **Description:** Contains 506 records with 14 features related to housing prices, including crime rate, number of rooms, tax rate, and location factors.

## Steps Performed

1. Data Loading and Preprocessing

2. Exploratory Data Analysis (EDA)

   * Checked data structure and summary statistics
   * Visualized correlations between features
   * Analyzed target variable distribution

3. Feature Selection

   * Selected relevant features affecting house prices

4. Model Building

   * Implemented Linear Regression
   * Implemented Decision Tree Regressor

5. Model Evaluation

   * Evaluated models using RMSE and MAE
   * Compared performance of both models

## Results

* Linear Regression provided stable and consistent predictions
* Decision Tree captured non-linear relationships
* RMSE used as primary evaluation metric
* Linear Regression performed better overall

## Tools Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Conclusion

This project demonstrates how machine learning can be applied to real-world problems like house price prediction. Linear Regression proved effective for this dataset, while Decision Tree provided additional insights into data patterns.
